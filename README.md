👾 Space Invaders – Консольный клон на 7 языках
Классическая аркада «Space Invaders» прямо в вашем терминале!
Защищайте Землю от вторжения пришельцев, стреляйте по врагам, набирайте очки и проходите уровни.
Реализована на семи языках программирования с единым геймплеем, цветной графикой, системой очков и рекордов, движущимися врагами, бонусами и настраиваемой сложностью.

🚀 Возможности
Игровой процесс – управляйте кораблём с помощью стрелок, стреляйте пробелом, уничтожайте волны врагов.

Враги – двигаются вправо-влево и постепенно опускаются вниз. С каждым уровнем скорость увеличивается.

Стрельба – лазеры летят вверх; попадание уничтожает врага (+10 очков).

Бонусы – при уничтожении врага иногда выпадает бонус: ускорение стрельбы или дополнительная жизнь.

Уровни – после уничтожения всех врагов начинается новый уровень с увеличенной скоростью.

Счёт и рекорд – текущий счёт и лучший результат сохраняются в локальный файл.

Цветная графика – корабль, враги, пули и бонусы окрашены в разные цвета.

Управление – стрелки влево/вправо для движения, пробел для стрельбы, Q для выхода, R для перезапуска.

Кроссплатформенность – работает в любом терминале с поддержкой ANSI (Linux, macOS, Windows с WSL или ConEmu).

📖 Использование
Синтаксис (единый для всех версий):

bash
<команда> [опции]
Опции
Опция	Описание
-s, --speed <N>	Базовая скорость игры (задержка в мс, по умолчанию 50)
-h, --help	Справка
Управление
Клавиша	Действие
← / A	Движение влево
→ / D	Движение вправо
Пробел	Стрельба
R	Перезапуск после окончания игры
Q	Выход из игры
🛠 Установка и запуск
Python
bash
pip install windows-curses  # только для Windows
python invaders.py [опции]
C++
bash
g++ -std=c++17 invaders.cpp -o invaders -lncurses
./invaders [опции]
Требуется библиотека ncurses (установите через пакетный менеджер).

Go
bash
go get github.com/nsf/termbox-go
go build invaders.go
./invaders [опции]
JavaScript (Node.js)
bash
npm install blessed
node invaders.js [опции]
C#
bash
csc invaders.cs
mono invaders.exe [опции]   # или dotnet run
Требуется поддержка ANSI в терминале (на Windows используйте Console.OutputEncoding = Encoding.UTF8).

Ruby
bash
ruby invaders.rb [опции]
Java
bash
javac -cp .:lanterna.jar invaders.java
java -cp .:lanterna.jar invaders [опции]
Требуется библиотека lanterna (скачайте JAR и добавьте в classpath).

🧠 Логика игры
Инициализация – корабль внизу, враги выстроены в ряды в верхней части экрана.

Движение врагов – все враги двигаются горизонтально, при достижении края экрана опускаются вниз и меняют направление.

Стрельба – при нажатии пробела выпускается лазер, который летит вверх; при попадании во врага он уничтожается.

Бонусы – с некоторой вероятностью после уничтожения врага появляется бонус (жизнь или ускорение).

Уровни – после уничтожения всех врагов начинается новый уровень с увеличенной скоростью.

Поражение – если враг достигает нижней границы или сталкивается с кораблём, игра заканчивается.

Счёт – начисляется за каждого уничтоженного врага; рекорд сохраняется.

✨ Дополнительные фичи
Анимированные враги – меняют цвет или символ при движении.

Звуковые эффекты – при выстреле и уничтожении врага (ASCII bell).

Пауза – нажмите P для приостановки игры.

Автосохранение – рекорд обновляется автоматически.
